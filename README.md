# MyZshTheme

## Installation

* Clone this repository into `$ZSH_CUSTOM/themes`

```shell
git clone https://github.com/Shemnei/my-zsh-theme.git ${ZSH_CUSTOM}/themes/mzt
```

* Activate the theme in `~/.zshrc`

```text
ZSH_THEME="mzt/mzt"
```

* To enable fish like path shrinking

```text
plugins=(shrink-path ...)
```

* Reload `~/.zshrc`

```shell
source ~/.zshrc
```
